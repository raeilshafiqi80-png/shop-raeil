## About the project
Nora Shop is a simple e-commerce fornt-end project built for learning web development.
it includes a home page, product section,slider and shopping cart.


## Features
-Beautiful Home page with background image
-product showcase
-Image Slider (Bootstrap Carousel)
-Shopping Cart system
-Dark mode/Light mode toggle
-Responsive design (mobile friendly)

## Technologies Used
-HTML5
-CSS3
-Bootstrap 5
-JavaScript

##How to Run the Project
1. Open the project in VS Code
2. Lnstall Live Server extension
3. Right click on index.html
4. Click Open with Live Server
